Quản Trị Hệ Thống

Công cụ Nhập liệu câu hỏi hàng loạt vào Firestore

📦 Bước 1: Chuẩn bị Dữ liệu

Để hệ thống đọc được dữ liệu, file Excel của bạn cần phải đúng định dạng cột quy định. Vui lòng tải file mẫu dưới đây về máy, điền câu hỏi của bạn vào và lưu lại.

🚀 Bước 2: Import Lên Hệ Thống

Chọn file Excel (.xlsx, .xls) mà bạn đã điền đầy đủ câu hỏi để đẩy lên cơ sở dữ liệu Firebase.

Kéo thả file vào đây hoặc Nhấn để chọn file
Chưa có file nào được chọn
Lưu ý: Quá trình import có thể mất vài giây tùy thuộc vào số lượng câu hỏi và tốc độ mạng. Vui lòng không đóng trình duyệt khi đang tải.
